# BFF (Backend for Frontend) Layer

- `web_bff.py` — ACTIVE in v0.1. Serves the React web dashboard.
- `desktop_bff.py` — SCAFFOLD. Planned for v2 (Electron/Tauri desktop app).
- `mobile_bff.py` — SCAFFOLD. Planned for v3 (React Native mobile app).

Each BFF adapts the core domain responses to the specific needs of each client surface.
